## Group

–í—ã –º–æ–∂–µ—Ç–µ –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞—Ç—å –º–∞—Ä—à—Ä—É—Ç—ã, —Å–æ–∑–¥–∞–≤ —Å—Ç—Ä—É–∫—Ç—É—Ä—É `*Group`.

**–ü—Ä–∏–º–µ—Ä**

```go
  v1 := api.Group("/v1", mysql())   // /api/v1

  v1.Get("/user", handler)          // /api/v1/user

  v2 := api.Group("/v2", mongodb()) // /api/v2
  v2.Get("/list", handler)          // /api/v2/list
  v2.Get("/user", handler)          // /api/v2/user

  app.Listen(3000)
}
```

## Listen

–ü—Ä–∏–≤—è–∑–∫–∏ –∏ –ø—Ä–æ—Å–ª—É—à–∏–≤–∞–Ω–∏–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–π –ø–æ —É–∫–∞–∑–∞–Ω–Ω–æ–º—É –∞–¥—Ä–µ—Å—É. –≠—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å `int` –¥–ª—è –ø–æ—Ä—Ç–∞ –∏–ª–∏ `string` –¥–ª—è –∞–¥—Ä–µ—Å–∞.

## Test

–¢–µ—Å—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ –≤–∞—à–µ–≥–æ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è –æ—Å—É—â–µ—Å—Ç–≤–ª—è–µ—Ç—Å—è –º–µ—Ç–æ–¥–æ–º **Test**. –ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ —ç—Ç–æ—Ç –º–µ—Ç–æ–¥ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è `_test.go` —Ñ–∞–π–ª–æ–≤ –∏–ª–∏ –∫–æ–≥–¥–∞ –≤–∞–º –Ω—É–∂–Ω–æ –æ—Ç–ª–∞–¥–∏—Ç—å –≤–∞—à—É –ª–æ–≥–∏–∫—É –º–∞—Ä—à—Ä—É—Ç–∏–∑–∞—Ü–∏–∏. –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é —Ç–∞–π–º-–∞—É—Ç `200 –º—Å`. –ï—Å–ª–∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ –ø–æ–ª–Ω–æ—Å—Ç—å—é –æ—Ç–∫–ª—é—á–∏—Ç—å —Ç–∞–π–º-–∞—É—Ç, —Ç–æ –ø–µ—Ä–µ–¥–∞–π—Ç–µ `-1` –≤ –∫–∞—á–µ—Å—Ç–≤–µ –≤—Ç–æ—Ä–æ–≥–æ –∞—Ä–≥—É–º–µ–Ω—Ç–∞.
